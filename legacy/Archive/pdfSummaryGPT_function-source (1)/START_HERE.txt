================================================================================
WHAT THIS DOES
================================================================================

This tool extracts startup names and founder names from PDF pitch decks.

Input:  Google Drive folder with PDFs
Output: CSV file with:
  - File name
  - Startup name
  - All founder names
  - Extraction method used

================================================================================
QUICK START (4 STEPS)
================================================================================

STEP 1: Install dependencies
  cd pdfSummaryGPT_function-source
  pip install -r requirements.txt

STEP 2: Get your Google Drive folder ID
  Open folder in Drive ‚Üí Copy ID from URL
  Example: https://drive.google.com/drive/folders/YOUR_ID_HERE

STEP 3: Run the script
  python batch_startup_extractor.py YOUR_FOLDER_ID

STEP 4: Check results
  Look for: startup_founders_data.csv in your temp folder

================================================================================
COMMANDS
================================================================================

Basic (fastest):
  python batch_startup_extractor.py 1a2b3c4d5e6f

With Gemini (more accurate):
  set GEMINI_API_KEY=your-api-key
  python batch_startup_extractor.py 1a2b3c4d5e6f --gemini

Custom output location:
  python batch_startup_extractor.py 1a2b3c4d5e6f --output ./results.csv

================================================================================
TROUBLESHOOTING
================================================================================

Problem: "0 PDFs found"
  ‚Üí Make sure folder has PDFs
  ‚Üí Check folder ID is correct

Problem: "ModuleNotFoundError"
  ‚Üí Run: pip install -r requirements.txt

Problem: Empty founders list
  ‚Üí Add --gemini flag for better detection
  ‚Üí Or PDFs don't have explicit founder mentions

Problem: "Permission denied"
  ‚Üí Use --output to save in different location
  ‚Üí Or check file permissions
